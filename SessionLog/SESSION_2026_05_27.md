  - `by=collection`: fetches collections + ShopTileImages + product counts per collection → `ShopByDirectory`
  - `by=category`: fetches top-level categories + ShopTileImages + product counts per category → `ShopByDirectory`

#### Data flow
- Brands: extracted from `attribute_lists.brand` on Medusa products (lightweight fetch with just `attribute_values` fields). Payload `Brands` collection provides logos matched by slug.
- Collections: `getCollections()` + `getShopTileImages()` + lightweight product fetch for count map by `collection.id`.
- Categories: `getCategories()` + `getShopTileImages()` + lightweight product fetch for count map by category id. Only top-level categories (no `parent_category_id`) shown.
- All items sorted by product count descending, then name ascending.
- "Featured" collection is hidden from the collection directory.
